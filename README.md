# thru 🧭

**Go thru the world, the right way.**

`thru` is an intelligent, beautifully crafted travel companion designed for the modern explorer who appreciates the charm of the past. It transforms travel planning from a chore into a curated journaling experience, blending advanced AI intelligence with a warm, neo-vintage "scrapbook" aesthetic.

---

## ✨ The Experience

### 🗺️ Intelligent Itinerary Architect
Plan your next expedition with a user-driven dashboard that gives you total control.
- **Live Preview**: See estimated costs and trip highlights update instantly as you adjust your preferences.
- **Curated Logic**: Filter by travel mode (Solo, Partner, Friends, Family) and interests (Culture, Food, Nature, etc.).
- **Quick Presets**: Jumpstart your journey with pre-optimized paths like _"Luxury Escape"_ or _"Budget Explorer."_

### 📖 Your Digital Scrapbook
Document your travels in a way that feels permanent.
- **The Journal**: A dedicated space to store photos, notes, and memories in a layout inspired by vintage travelogues.
- **Interactive Maps**: Visualize your global footprint and the paths you've taken.

### 🛂 The Global Passport
Gamify your exploration and build your legacy.
- **Digital Stamps**: Earn unique stamps for every destination you visit.
- **Badges & Awards**: Unlock achievements as you discover hidden gems and complete expeditions.

### 🎨 Neo-Vintage Aesthetic
The interface is designed to feel like a well-worn travel journal:
- **Paper Textures**: Subtle noise and linen filters for a tactile digital feel.
- **Typewriter Typography**: Special Elite fonts and stamp effects for that authentic "manifest" look.
- **Harmonious Palette**: Warm paper tones (`#F5EBDD`), deep ink (`#3B2F2F`), and classic stamp red (`#A44A3F`).

## 🛠️ Technology Stack

- **Frontend**: React (v19) + Vite
- **Styling**: Tailwind CSS + Framer Motion for micro-animations
- **Backend**: Node.js + Express (Server-side rendering & API middleware)
- **Database/Auth**: Firebase & Firestore
- **Intelligence**: Google Gemini AI (for itinerary generation and planning assistance)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- A Gemini API Key (for AI features)

### Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/saeeekumbhar/thru.git
   cd thru
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   Create a `.env` file in the root directory and add your keys:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Launch the Expedition**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.
